## Install using Composer

### For basic settings

1.  Install and activate the plugin with the plugin manager.
2.  Reload the backend.
3.  Go to _Configuration > Payment methods_. Additional payment methods have been added with the prefix “cko” and you can now find the Checkout.com _Setup menu_ point.

If you want to see the activated payment methods in the frontend, you need to add your public and private key from [Checkout.com](http://checkout.com/) to the configuration.

<TipBox variant="note">

In order to show Apple Pay, Google Pay, SEPA, credit card and PayPal in the front-end you need to configure the according payment method in the Checkout Setup.

</TipBox>

### For Apple Pay

<TipBox variant="info">

See our [Apple Pay documentation](/payments/payment-methods/wallets/apple-pay) for more information.

</TipBox>

#### Checkout setup:

1.  Navigate to the Apple Pay tab.
2.  Fill out the required fields and select _Save._

#### Domain verification:

1.  Navigate to the Domain Verify tab.
2.  Edit the path to match the path give by Apple Pay.
3.  Upload the verification file from your Apple Pay Developer account.
4.  Verify the upload by selecting _OK_ in your Apple Pay Developer account.

#### Certification Upload

1.  Create and download the Apple Pay Payment Processing Certification from your Apple Pay Developer account.
2.  Upload the certificate to the Shopware Backend and select _CSR Certificate._
3.  The PEM certification will be needed to validate your merchant identification session.
