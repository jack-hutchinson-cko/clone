<IBuilderDescriptionCard title="Add validation event listener" tab="app.js" lines={[17, 21]}>

Frames creates its own events to detect the payment method, whether the form is valid and more. To access these events, use `Frames.Events.<EVENT_NAME>`. See the [Frames reference](/integrate/frames/frames-reference) to learn about the events and what they do.

First, weâ€™ll focus on form validation. Use the `FRAMES_VALIDATION_CHANGED` event to listen out for field validation status changes. The second argument will be a function that checks if the card is valid. To do this, we can use a built-in action called `isCardValid()`, which returns a Boolean value.

You can also create a second event listener on the validation event that handles error message styling. 

```js
//Frames event for validation change
Frames.addEventHandler(
   Frames.Events.FRAME_VALIDATION_CHANGED,
   onValidationChanged
);
```
