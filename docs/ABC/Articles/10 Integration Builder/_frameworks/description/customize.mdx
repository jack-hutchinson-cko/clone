You can customize the language, CSS and placeholders on input fields in the iframes. Do this when initializing Frames by specifying options as another argument. For example:

```js
Frames.init(
‘pk_test_xxx’, {
{
	localization: "FR-FR",
	style: {
		base: {
			color: "black",
			fontSize: "18px",
		},
	},
)
```
